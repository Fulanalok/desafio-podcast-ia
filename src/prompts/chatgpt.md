# Prompts para ChatGPT

Prompts usados para criar e revisar o roteiro do podcast.

| Etapa | Prompt |
| :--- | :--- |
| Tema | Quero criar um podcast curto para estudantes iniciantes sobre inteligencia artificial. Sugira 5 temas simples, atuais e faceis de explicar em ate 3 minutos. O tom deve ser didatico, leve e motivador. |
| Titulo | Com base no tema "engenharia de prompts para iniciantes", crie 10 opcoes de titulo para um podcast curto. Os titulos devem ser chamativos, simples e adequados para um publico iniciante em tecnologia. |
| Estrutura | Crie uma estrutura para um episodio de podcast de ate 3 minutos com o titulo "Prompts na Pratica: como conversar melhor com a IA". A estrutura deve conter abertura, apresentacao do problema, explicacao principal, exemplo pratico, conclusao e chamada final. |
| Roteiro | Escreva um roteiro de podcast em portugues do Brasil, com linguagem simples e tom amigavel. Explique o que e um prompt, por que prompts claros melhoram respostas de IA, quais elementos ajudam em um bom prompt, um exemplo de prompt fraco e um exemplo melhorado. |
| Revisao | Revise o roteiro para ficar mais natural em formato de podcast. Mantenha linguagem simples, frases curtas e ritmo oral. Evite termos muito tecnicos e deixe a explicacao facil para iniciantes. |
