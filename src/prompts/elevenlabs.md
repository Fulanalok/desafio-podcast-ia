# Direcao para ElevenLabs

Texto e configuracao sugeridos para gerar o audio no ElevenLabs.

## Texto

Use o roteiro final em:

```text
roteiro-podcast.md
```

## Direcao de voz

```text
Narrar em portugues do Brasil, com tom didatico, amigavel e natural.
O ritmo deve ser calmo, mas nao lento demais.
Usar pausas curtas entre blocos de ideias.
Dar leve enfase em termos como "prompts", "contexto", "objetivo" e "revisao humana".
Soar como uma explicacao de professor para alunos iniciantes.
```

## Configuracao sugerida

```text
Stability: 45% a 60%
Similarity: 70% a 85%
Style exaggeration: baixo ou medio
Speaker boost: ligado, se disponivel
Formato de saida: MP3
```

## Arquivo final

Depois de baixar o audio do ElevenLabs, salve em:

```text
output/podcast_editado.mp3
```

Se quiser manter uma copia secundaria, tambem salve em:

```text
audio/podcast-prompts-na-pratica.mp3
```
