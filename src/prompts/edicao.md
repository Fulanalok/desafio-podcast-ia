# Prompt para Edicao e Revisao Final

Prompt usado para orientar a revisao do audio e da entrega.

```text
Analise este projeto de podcast criado com IA e sugira melhorias para entrega na DIO.

Verifique:
- clareza do tema;
- naturalidade do roteiro;
- organizacao dos prompts;
- presenca do arquivo de audio;
- existencia de capa;
- README com explicacao do processo;
- referencia ao repositorio base do desafio.

O objetivo e deixar o projeto com cara de portifolio, sem parecer apenas uma resposta automatica.
```

Checklist aplicado:

- roteiro curto e direto;
- audio salvo no repositorio;
- prompts documentados;
- capa adicionada;
- README com preview e explicacao do processo;
- estrutura inspirada no repositorio base.
