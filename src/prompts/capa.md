# Prompt para Capa

Prompt usado como direcao visual para a capa do podcast.

```text
Crie uma capa para um podcast chamado "Prompts na Pratica".

Tema: inteligencia artificial para iniciantes.
Estilo: moderno, limpo e tecnologico.
Elemento principal: microfone.
Elementos secundarios: formas abstratas que lembrem conversa, dados ou IA.
Cores: azul escuro, verde e branco.
Texto na capa: "Prompts na Pratica" e "Como conversar melhor com a IA".

A imagem deve funcionar bem como capa quadrada ou banner de repositorio.
```

Observacao: neste repositorio, a capa final foi criada em SVG para manter o arquivo leve e facil de versionar.
