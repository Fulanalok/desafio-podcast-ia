# Prompts Utilizados

Este arquivo documenta os prompts utilizados para criar o podcast do desafio.

## 1. Prompt para definir o tema

```text
Quero criar um podcast curto para estudantes iniciantes sobre inteligencia artificial.
Sugira 5 temas simples, atuais e faceis de explicar em ate 3 minutos.
O tom deve ser didatico, leve e motivador.
```

## 2. Prompt para escolher o titulo

```text
Com base no tema "engenharia de prompts para iniciantes", crie 10 opcoes de titulo para um podcast curto.
Os titulos devem ser chamativos, simples e adequados para um publico iniciante em tecnologia.
```

## 3. Prompt para estruturar o episodio

```text
Crie uma estrutura para um episodio de podcast de ate 3 minutos com o titulo:
"Prompts na Pratica: como conversar melhor com a IA".

A estrutura deve conter:
- abertura;
- apresentacao do problema;
- explicacao principal;
- exemplo pratico;
- conclusao;
- chamada final.
```

## 4. Prompt para gerar o roteiro

```text
Escreva um roteiro de podcast em portugues do Brasil, com linguagem simples e tom amigavel.

Tema: engenharia de prompts para iniciantes.
Titulo: "Prompts na Pratica: como conversar melhor com a IA".
Duracao aproximada: 3 minutos.

O roteiro deve explicar:
- o que e um prompt;
- por que prompts claros melhoram respostas de IA;
- quais elementos ajudam em um bom prompt;
- um exemplo de prompt fraco e um exemplo de prompt melhorado;
- uma conclusao motivadora.
```

## 5. Prompt para revisar o roteiro

```text
Revise o roteiro abaixo para ficar mais natural em formato de podcast.
Mantenha linguagem simples, frases curtas e ritmo oral.
Evite termos muito tecnicos e deixe a explicacao facil para iniciantes.
```

## 6. Prompt para gerar descricao do projeto no GitHub

```text
Crie uma descricao curta para um repositorio GitHub de um projeto de podcast gerado com IA.
Inclua objetivo, arquivos do projeto, conceitos usados e instrucoes de entrega.
```

## 7. Prompt para capa do podcast

```text
Crie uma ideia de capa para um podcast chamado "Prompts na Pratica".
O tema e inteligencia artificial para iniciantes.
A capa deve transmitir tecnologia, clareza e aprendizado.
Use um microfone como elemento principal e uma paleta com azul escuro, verde e branco.
```

## 8. Prompt para direcao de audio

```text
Transforme este roteiro em uma narracao natural para podcast.
Use pausas curtas entre blocos de ideias.
O tom deve ser didatico, calmo e proximo.
Evite velocidade muito rapida e destaque os termos principais.
```

## 9. Prompt para revisao final do repositorio

```text
Revise a estrutura de um repositorio GitHub para entrega de um podcast criado com IA.
Verifique se existem README, prompts usados, roteiro, arquivo de audio, capa e referencia ao projeto base.
Sugira melhorias para deixar o projeto mais claro para avaliadores.
```
